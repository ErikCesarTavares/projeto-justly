## ✨ Projeto

O site **Justly** foi criado para oferecer uma interface elegante e funcional, com foco em destacar os serviços do escritório de advocacia. Suas principais funcionalidades incluem:

1. **Design Responsivo**: Totalmente adaptável para dispositivos móveis, tablets e desktops.
2. **Navegação Intuitiva**: Menu acessível e interativo para explorar as páginas.
3. **Seções Dinâmicas**:
   - Apresentação com imagem de fundo e texto destacado.
   - Áreas de serviços detalhadas.
   - Formulário de consulta para solicitações de orçamento ou agendamentos.
4. **Estilo Moderno**: Uso de fontes estilizadas e esquema de cores profissionais para uma aparência atraente.
